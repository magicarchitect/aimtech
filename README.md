# Aimtech — Web corporativa

Web estática para www.aimtech.es. Migración desde WordPress.

## Stack
- HTML5 + CSS3 vanilla (variables CSS, sin framework)
- JS vanilla
- Fuentes: Instrument Serif, JetBrains Mono, Geist (Google Fonts)

## Desarrollo
Abrir `index.html` con Live Server (extensión VS Code) o:
npx serve .
