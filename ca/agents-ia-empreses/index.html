Agents d'IA per a empreses.
Que fan la feina, amb criteri.

Un agent d'IA és un assistent que executa tasques per tu —triatge de correus, primera atenció, extracció de documents, seguiment— i escala a una persona quan toca. El dissenyem, el connectem a les teves eines i el posem en producció amb human-in-the-loop i ROI mesurable. Sense agents autònoms que decideixen sols el que no han de decidir.

01 — Què fa un agent

No és un chatbot. Executa tasques / de principi a fi, / no només respon.

Un agent rep un activador, decideix què fer, fa servir les teves eines i deixa la feina feta (o preparada perquè una persona l'aprovi). Aquestes són les quatre famílies que més es demanen.

// agent_01 Correus

Triatge i primera resposta.

Llegeix la safata, classifica per tipus, redacta una primera resposta a les consultes freqüents i deriva a l'àrea correcta. El que és sensible es queda en esborrany perquè una persona l'aprovi.

// agent_02 Atenció

Atenció web i WhatsApp.

Respon preguntes freqüents sobre la teva documentació, capta el lead, agenda o deriva a l'humà quan la conversa ho demana. Amb to ajustat al de la teva marca, no robòtic.

// agent_03 Documents

Llegir i extreure.

Treu dades de factures, contractes o formularis, les valida contra les teves regles i les dona d'alta al sistema. Consulta la teva base documental (RAG) per respondre amb fonts.

// agent_04 Seguiment

Vigilar i avisar.

Fa seguiment de tasques, terminis o indicadors i avisa quan alguna cosa se surt de rang o requereix acció. L'equip deixa de vigilar manualment el que l'agent vigila sol.

02 — Amb persona en el bucle

Un agent no hauria de decidir-ho tot / sol. / I el nostre no ho fa.

La diferència entre una demo i una cosa que pots posar en producció és el control. Les accions sensibles passen per una persona; les rutinàries s'executen soles i s'auditen. Aprofundim en el perquè a les tres preguntes abans d'implantar IA agèntica.

// control_01 Aprovació

El que és crític, a revisió.

Enviaments externs, decisions de cobrament o publicacions passen per una safata d'aprovació abans d'executar-se. Tu decideixes què pot fer sol i què no.

// control_02 Traçabilitat

Tot queda registrat.

Cada pas de l'agent deixa traça: què va rebre, què va decidir, què va fer. Auditable i reversible quan té sentit. Sense caixes negres.

// control_03 ROI

Es mesura, o no va.

Abans de començar fixem la línia base i el llindar de millora. Si l'agent no aporta davant de fer-ho a mà, s'ajusta o es para. Sense lletra petita.

03 — Contractació

Per a empreses, entitats i AAPP.

Cada agent es cotitza per abast després d'un diagnòstic curt: no hi ha dos casos iguals. Projectes des de 3.500€; l'abast defineix el pressupost final. Si el teu cas és més senzill —automatitzar un procés sense agents—, mira l'automatització de processos amb IA; i el mètode complet d'orquestració és a implantació d'IA agèntica.

// segment_01 Empreses

Un agent en producció.

Disseny, connexió a les teves eines, human-in-the-loop i mesura de ROI. Comencem per un agent i un procés, no per una plataforma enorme.
